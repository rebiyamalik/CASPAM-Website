Computer Centre

The Computer Centre at CASPAM serves as a hub for computational research, data analysis, and scientific computing. It is equipped with advanced hardware, software and networking infrastructure to support advanced mathematical modeling, simulation, and analysis.

Programming Lab

  • Desktop computers with scientific and numerical software
  • Projector and audio-visual aids
  • Internet access and learning management integration

Networking Lab

  1. Routers and switches for hands-on networking labs
  2. Structured cabling and LAN setup practice
  3. Network monitoring and simulation tools
